pico-bs demo

Purpose

Pico-bs brings extra Bootstrap-like functionality and stylings to the Pico CSS framework. It adds missing BS utility classes, simplified grid/flexbox/column layouts, and declarative components like tabs, dialogs, alerts and tooltips. All without javascript.

columns

Provides a simple multi-column layout using the [data-pbs-cols] attribute.

options
[data-pbs-cols] Sets the numner of columns, default is 3.
[data-pbs-gap] Sets the column-gap (aka gutter) with a CSS length unit, default is 0.5rem
[data-pbs-width] Sets the column-width with a CSS length unit.

.center Align children in the middle and center
.columns Force the flexbox to a vertical orientation
.reverse Reverses the order of child items

Popovers

3 Steps to adding popovers
  1. Button with popovertarget attrib pointing to the id of the popup content, and a popovertargetaction attrib of show | hide | toggle.
  2. Content with an id attrib to recieve button clicks, an empty popover attrib, and a data-pbs-popto attrib, always starting with --, pointing to the identifier of the target element to be decorated by this content.
  3. Target element, like a form input or code snipet, with a data-pbs-pop attribute with the popover identifier, always starting with --.

Add a handy tooltip class to style the popover to look like a tooltip.
