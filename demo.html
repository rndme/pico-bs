pico-bs demo

Purpose

pico-bs brings extra bootstrap-like functionality and stylings to the picocss framework. It adds handy utlity classes, interactive widgets, declarative components like tabs, choosers, alerts and tooltips. Most features are implimented in CSS, while JS is used sparingly to glue concepts and provide interaction.

Chooser

Use a chooser class to define a chooser-style content container built only from content elements, using the name or data-name attribute of the child elements to label the tab handle. Optionally use the data-pbs-chooser-menu-width attrib to specify a css length for the handle width.

columns

Provides a simple multi-column layout using the [data-pbs-cols] attribute.

options
[data-pbs-cols] Sets the numner of columns, default is 3.
[data-pbs-gap] Sets the column-gap (aka gutter) with a CSS length unit, default is 0.5rem
[data-pbs-width] Sets the column-width with a CSS length unit.

flex (content determines size of columns and rows)

Flexbox is enabled with the .flex class. By default, the child items are evenly spaced columns. You can alter that with a few additional classes:


.center Align children in the middle and center
.columns Force the flexbox to a vertical orientation
.reverse Reverses the order of child items

Filtering

Used on an input, data-pbs-filter-items defines selectors pointing to other elements to show/hide based on matching a search term. Use in conjuction with data-pbs-filter-property to determine the property of the target elements from which to obtain the text to search, default is textContent.
